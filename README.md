# Pseudo3DDisplayBoxWithHeadTracking_Scripts
裸眼立体ディスプレイのUnityC#スクリプトです。  
制作期間 2021年5月～6月(修士1年生)  
VIVE Trackerで頭部を追跡して、運動視差で立体のように見せます。  
はじめは研究のネタ探しの一環として制作していましたが、pCubeeという存在が海外論文にあるのを知り、あまりにも同じだったためボツになりました。
